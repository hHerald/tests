# 📊 Портфолио решений тестовых заданий по аналитике данных

[![Python](https://img.shields.io/badge/Python-3.12-blue?logo=python)](https://www.python.org/)
[![Jupyter](https://img.shields.io/badge/Jupyter-Notebook-orange?logo=jupyter)](https://jupyter.org/)
[![Pandas](https://img.shields.io/badge/Pandas-Analysis-150458?logo=pandas)](https://pandas.pydata.org/)
[![Power BI](https://img.shields.io/badge/Power%20BI-Desktop-yellow?logo=powerbi)](https://powerbi.microsoft.com/)

Репозиторий содержит решения реальных тестовых заданий и практических кейсов по анализу данных, продуктовой, операционной аналитике и визуализации в Power BI. Каждое задание выполнено с нуля: от разведочного анализа до формулировки метрик, проверки гипотез, построения моделей и создания интерактивных дашбордов.

## 🎯 Цель репозитория

Демонстрация навыков:
- Разведочного анализа данных (EDA) и визуализации
- Формулирования и расчёта бизнес-метрик
- Статистического тестирования гипотез
- Машинного обучения и анализа важности признаков
- Работы с реальными операционными процессами
- Создания профессиональных дашбордов в **Power BI**
- Подготовки результатов для менеджеров и стейкхолдеров
- Анализа API маркетплейсов и подготовки технических заданий
- Анализа воронок конверсии и продуктовой аналитики
- Финансового анализа рентабельности продаж на маркетплейсах

## 🛠️ Используемые технологии

- **Python**: `pandas`, `numpy`, `scipy`, `scikit-learn`, `statsmodels`, `requests`
- **Визуализация**: `matplotlib`, `seaborn`
- **Power BI**: Power BI Desktop, DAX (базовый), интерактивные дашборды, карты, ABC-анализ, визуализация воронок, финансовые KPI и геоанализ
- **API и данные**: Wildberries API, работа с отчётами реализации и остатками
- **Инструменты**: Jupyter Notebook, работа с реальными датасетами и API маркетплейсов

## 🚀 Как изучить решения

1. Перейдите в папку нужного кейса
2. Для Python-кейсов — откройте Jupyter Notebook
3. Для Power BI — откройте файл `.pbix` в **Power BI Desktop**
4. Ознакомьтесь с локальным `README.md` внутри папки
