# Business Development Analytics Test Assignment

Тестовое задание от команды аналитики по развитию бизнеса.

## Описание

В репозитории представлено решение тестового задания, включающего задачи по SQL, анализу данных и бизнес-аналитике.

Рассмотренные темы:

* анализ причин возвратов товаров;
* оценка влияния очередей в пунктах выдачи на выкуп заказов;
* поиск причин падения продаж бренда;
* определение и разметка оттока клиентов.
