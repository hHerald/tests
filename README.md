# 📊 Портфолио решений тестовых заданий по аналитике данных

[![Python](https://img.shields.io/badge/Python-3.12-blue?logo=python)](https://www.python.org/)
[![Jupyter](https://img.shields.io/badge/Jupyter-Notebook-orange?logo=jupyter)](https://jupyter.org/)
[![Pandas](https://img.shields.io/badge/Pandas-Analysis-150458?logo=pandas)](https://pandas.pydata.org/)

Репозиторий содержит решения реальных тестовых заданий и практических кейсов по анализу данных, продуктовой и операционной аналитике. Каждое задание выполнено с нуля: от разведочного анализа до формулировки метрик, проверки гипотез и подготовки выводов, понятных для бизнеса.

## 🎯 Цель репозитория

Демонстрация навыков:
- Разведочного анализа данных (EDA) и визуализации
- Формулирования и расчёта бизнес-метрик
- Статистического тестирования гипотез
- Машинного обучения и анализа важности признаков
- Работы с реальными операционными процессами
- Подготовки результатов для менеджеров и стейкхолдеров

## 📁 Выполненные кейсы

| Папка | Описание | Основные навыки | Статус |
|-------|----------|------------------|--------|
| **Практика 3 курс. Анализ Youtube Shorts** | Анализ 300 коротких видео YouTube. Построение модели предсказания коэффициента вовлечённости (`engagement_rate`) | EDA, визуализация, статистика, Gradient Boosting, Feature Importance | ✅ Выполнено |
| **Тестовое задание. Аналитика по развитию бизнеса** | Анализ возвратов товаров, очередей в ПВЗ, падения продаж бренда, определение и разметка оттока клиентов | SQL, Product Analytics, Cohort & Churn Analysis, формулирование гипотез | ✅ Выполнено |
| **Тестовое задание. Операционная дирекция** | Анализ ёмкости и доступности слотов курьерской доставки (Ламода). Выявление и классификация проблемных слотов | Operations Analytics, расчёт метрик доступности, классификация проблем, pandas | ✅ Выполнено |

## 🛠️ Используемые технологии

- **Python**: `pandas`, `numpy`, `scipy`, `scikit-learn`, `statsmodels`
- **Визуализация**: `matplotlib`, `seaborn`
- **Инструменты**: Jupyter Notebook, работа с реальными датасетами

## 🚀 Как изучить решения

1. Перейдите в папку нужного кейса
2. Откройте Jupyter Notebook
3. Запустите ячейки по порядку
4. Ознакомьтесь с локальным `README.md` внутри папки — там подробное описание и выводы

## 💡 Что особенно ценно в этом портфолио

- Умение **формулировать полезные бизнес-метрики**
- Навык **выявления и классификации проблем** (особенно ярко проявлено в операционном кейсе)
- Чистый код + качественная документация
- Результаты, ориентированные на принятие решений, а не просто на красивые графики
