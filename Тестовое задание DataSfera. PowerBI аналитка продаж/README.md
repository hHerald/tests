# 📊 Power BI Отчет: Заказы с минусовой прибылью

**Название отчета:** `test_DSf.pbix`  
**Тема:** Анализ убыточных заказов и эффективности продаж

## 🎯 Цель отчета

Выявить и проанализировать заказы с отрицательной прибылью, определить проблемных менеджеров, сегменты и категории товаров, а также оценить географическое распределение продаж и ABC-сегментацию клиентов.

Отчет помогает операционному и коммерческому руководству понять причины убытков и принять меры по повышению рентабельности.

## 🛠️ Техническая информация

- **Файл:** `test_DSf.pbix`
- **Формат экспорта:** Power BI Desktop
- **Тема:** Современная тема Power BI (CY26SU05 + AccessibleDefault)
- **Визуализации использованы:**
  - Таблицы
  - Карточки (KPI)
  - Горизонтальные бар-чарты
  - Карта (Map / Azure Map)
  - Столбчатая диаграмма динамики
  - Кольцевые/круговые диаграммы (сегменты)

## 📌 Как открыть и использовать отчет

1. Установите **Power BI Desktop** (бесплатно)
2. Откройте файл `test_DSf.pbix`
3. При необходимости обновите данные (если подключены к источнику)
4. Используйте срезы (slicers), если они добавлены, для фильтрации
5. Наводите курсор на визуализации для получения подсказок (tooltips)

## 💡 Ключевые выводы и рекомендации (пример)

- Существует значительная проблема с убыточными заказами у нескольких менеджеров (особенно топ-3).
- Необходимо провести анализ причин отрицательной прибыли (возможно, высокие скидки, логистика или возврат товаров).
- Сегмент **A** приносит основную прибыль, несмотря на меньшее количество клиентов.
- Географическое распределение показывает концентрацию продаж в определённых провинциях Канады.
- Рекомендуется:
  - Провести индивидуальную работу с менеджерами из топа убытков
  - Пересмотреть ценообразование и условия по убыточным категориям
  - Усилить контроль над логистическими затратами
