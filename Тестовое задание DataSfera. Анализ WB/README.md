# 📊 Power BI Отчет: Анализ финансов и продаж Wildberries (FBS)

**Название отчета:** `analyse_WB.pbix`  
**Тема:** Финансовый анализ рентабельности продаж пуховых платков и перчаток на маркетплейсе Wildberries. Выявление лидеров роста/падения, географии продаж и возвратов.

## 🎯 Цель отчета

Создать автоматизированный и наглядный инструмент для оценки реальной стоимости продаж на WB (логистика, комиссия WB, налоги, возвраты). 

Отчет помогает собственнику и закупщикам:
- Понять, насколько рентабельны продажи в текущий период
- Быстро выявлять артикулы-лидеры роста и проблемные позиции (лидеры падения)
- Анализировать географическое распределение продаж для приоритизации складов и допоставок
- Отслеживать возвраты и их влияние на финансы
- Принимать обоснованные решения по ассортименту, ценообразованию и логистике

**Решаемые боли (согласно ТЗ):**
- Непонимание реальных затрат на продажу (логистика + комиссия)
- Сложность оценки, какие товары перестают покупать
- Отсутствие понимания, какие регионы/склады в приоритете

## 🛠️ Техническая информация

- **Файл:** `analyse_WB.pbix`
- **Размер:** ~18,9 МБ
- **Формат:** Power BI Desktop (версия с темой CY26SU05)
- **Источники данных:**
  - Wildberries API (БД Postgres): Поставки, Продажи, Отчёт о продажах по реализации, Склад, Заказы
  - Google Docs (таблицы): Таблица себестоимости, Справочник номенклатуры (артикулов), Тарифы доставки
- **Рассчитанные меры:** Выручка, Комиссия, Логистика, Валовая прибыль, Рентабельность, ROI, Количество заказов
- **Визуализации:** Карточки (card / multi-row card), Линейные графики, Горизонтальные бар-чарты, Кнопки навигации, Срезы (slicers)
- **Периодичность обновления:** Ежемесячно (или по требованию при обновлении данных из API)

## 📌 Как открыть и использовать отчет

1. Установите **Power BI Desktop** (бесплатно)
2. Откройте файл `analyse_WB.pbix`
3. При первом открытии обновите данные (**Home → Refresh**), если подключены к Postgres / Google Sheets
4. Используйте верхние фильтры по датам и артикулам
5. Переключайтесь между разделами через кнопки навигации
6. Для детального анализа кликайте на бары и линии — сработает кросс-фильтрация
7. Для отправки руководству: **File → Export → PDF** или опубликуйте в Power BI Service

## 💡 Ключевые выводы и рекомендации

### Критические проблемы (требуют немедленного внимания)
- **Глубокая убыточность:** Рентабельность **-125,58%** и валовая прибыль **-70 млн ₽**. Логистика (13,56 млн) и комиссия (5,45 млн) «съедают» всю выручку.
- **ROI на отрицательном уровне** весь период — продажи в текущей модели не окупаются.
- **Высокие возвраты** у самых продаваемых артикулов (особенно supplierArticle 96 и 87).

### Рекомендации собственнику и закупщикам

1. **Срочно пересмотреть ценообразование** — текущая маржа не покрывает даже прямые затраты WB.
2. **Проанализировать топ-3 падающих артикула** (223, 120, 207) — возможно, вывести из ассортимента или изменить позиционирование.
3. **Проверить качество и описания** у лидеров возвратов (96, 87, 93). Высокий % возвратов = доп. расходы на логистику в обе стороны.
4. **Сфокусироваться на Москве и МО** — здесь максимальная концентрация продаж. Усилить presence на этих складах.
5. **Подключить данные о себестоимости** из Google-таблиц (ещё не интегрированы в текущей версии) — без этого точный расчёт чистой прибыли невозможен.
6. **Внедрить еженедельный мониторинг ROI** и алерты при падении ниже -0,5.
7. **Рассмотреть изменение условий FBS** или частичный переход на другие схемы поставок, если логистика остаётся убыточной.
